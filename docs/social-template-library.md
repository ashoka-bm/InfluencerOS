# Social Template Library

InfluencerOS uses templates after an idea is selected. The idea says what the post is about. The template says how the viewer moves through it.

## Starter Set

### Short-Form Video

- `template_hook_problem_solution`: hook -> problem -> solution -> payoff.
- `template_constraint_countdown_result`: constraint -> timer/countdown -> action -> result.
- `template_before_process_payoff`: before state -> process -> visible payoff.

### Carousel

- `template_hook_steps_payoff`: cover hook -> step sequence -> payoff/save cue.
- `template_myth_truth_slides`: myth -> correction -> proof/example -> takeaway.
- `template_before_after_breakdown`: before -> after -> what changed -> how to try it.

### Single Image Post

- `template_identity_signal`: one visual signal that reinforces who the creator is.
- `template_quote_visual`: concise statement with strong visual styling.
- `template_avatar_moment`: avatar in a memorable situation that communicates the niche.

### Story Sequence

- `template_day_fragment_sequence`: quick fragments of a day or routine.
- `template_bts_mini_arc`: behind-the-scenes setup -> moment -> small reveal.
- `template_teaser_reveal`: tease -> partial context -> reveal.

## Rule

Templates must be compatible with the recommended format. Do not apply a video-only motion structure to a carousel unless each movement can become a slide-level visual beat.
